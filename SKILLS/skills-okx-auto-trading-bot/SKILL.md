# OKX 自动交易机器人

## 支持的4种策略

| 策略 | 说明 | 适用信号 | 需要参数 |
|------|------|----------|----------|
| 网格 | 震荡行情，低买高卖 | MA5≈MA10，差距<3% | 网格数、价格区间、金额 |
| 定投 | 定期定额买入 | MA5>MA10，MACD金叉 | 每次金额、间隔 |
| DCD双币 | 持有币种理财 | 波动率>15% | 理财周期 |
| 马丁格尔 | 亏损翻倍加仓 | MACD死叉，RSI<50 | 基础金额、翻倍倍数、最大层数 |

## 策略判断逻辑

### 网格策略
- 适用条件：MA5 与 MA10 差距 < 3%
- 推荐理由：价格在区间内震荡，低买高卖赚取波动收益

### 定投策略
- 适用条件：MA5 > MA10 + MACD金叉 + RSI < 70
- 推荐理由：上涨趋势中定投，长期持有享受涨幅

### DCD双币策略
- 适用条件：波动率 > 15%
- 推荐理由：高波动币种理财收益高

### 马丁格尔策略
- 适用条件：MACD死叉 + RSI < 50 + 下跌趋势
- 推荐理由：超卖后反弹概率大，翻倍加仓快速回本盈利

## 下单流程

### 第一步：分析行情
```
1. 获取币种行情和技术指标
2. 判断适合的策略类型（网格/定投/DCD/马丁格尔）
3. 显示分析结果，询问用户是否要下单
```

### 第二步：询问参数
用户确认要下单后，根据策略类型询问参数：

**网格策略：**
```
请设置网格参数：
- 网格数量（默认5）：？
- 价格区间%（默认10%）：？
- 投入金额（USDT）：？
```

**定投策略：**
```
请设置定投参数：
- 每次投入金额（USDT）：？
- 间隔小时数（默认24）：？
```

**马丁格尔策略：**
```
请设置马丁格尔参数：
- 基础金额（USDT，默认1）：？
- 翻倍倍数（默认2）：？
- 最大层数（默认3）：？
```

**DCD双币策略：**
```
请设置DCD参数：
- 理财周期（天，默认7）：？
- 投入金额（USDT）：？
```

### 第三步：确认下单
参数确定后，显示完整订单信息让用户确认：

```
📝 订单确认

币种：XRP
策略：定投
理由：MA5>MA10+MACD金叉，RSI=51

参数：
- 买入金额：3 USDT
- 可买数量：2.16 XRP
- 买入价格：市价
- 止盈：+10%
- 止损：-10%

确认请回复"确认"
回复"否"取消
```

### 第四步：执行下单
用户确认后，执行下单并设置止盈止损。

## 常用命令

| 用户回复 | 操作 |
|---------|------|
| 网格 | 推荐网格策略 |
| 定投 | 推荐定投策略 |
| DCD/双币 | 推荐DCD策略 |
| 马丁格尔 | 推荐马丁格尔策略 |
| 否 | 取消下单 |
| 确认 | 确认下单 |
| 停止策略 | 全部平仓 |
| 收益 | 查询持仓盈亏 |

## OKX API 调用

```bash
# 查询余额
okx account balance

# 市价买入
okx spot place --instId XRP-USDT --side buy --ordType market --sz 数量

# 止盈止损
okx spot algo place --instId XRP-USDT --side sell --sz 数量 --ordType oco --tpTriggerPx=止盈价 --slTriggerPx=止损价
```
